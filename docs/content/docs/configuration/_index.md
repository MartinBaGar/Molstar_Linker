The **Mol** Linker Studio\* is the full settings page. Open it via the popup or your browser's extension options.

It has four panels:

-   [Scene Settings](./scene) — background color and camera
-   [Molecular Targets](./targets) — representations per molecule type
-   [Custom Rules](./custom-rules) — per-chain, per-residue highlights and overlays
-   [Custom Domains](../custom-domains) — authorize private GitLab instances
