    Configuration

    The Mol Linker Studio* is the full settings page. Open it via the popup or your browser’s extension options.

    It has four panels:

    • Scene Settings — background color and camera
    • Molecular Targets — representations per molecule type
    • Custom Rules — per-chain, per-residue highlights and overlays
    • Custom Domains — authorize private GitLab instances
