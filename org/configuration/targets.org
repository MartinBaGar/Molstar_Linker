The *Targets* section in the Options page lets you configure the default visual style for each molecular component. Each target is represented as a collapsible card.

* Available Targets

| Target                    | Selector   | Default Representation | Default Color      | Default Size | Default Opacity |
|---------------------------+------------+------------------------+--------------------+--------------+-----------------|
| Proteins                  | =protein=  | Cartoon                | Chain ID           | —            | 1.0             |
| Nucleic Acids (DNA/RNA)   | =nucleic=  | Cartoon                | Chain ID           | —            | 1.0             |
| Ligands & Small Molecules | =ligand=   | Ball & Stick           | Element symbol     | 0.2          | 1.0             |
| Carbohydrates & Glycans   | =branched= | Carbohydrate           | Chain ID           | —            | 1.0             |
| Lipids                    | =lipid=    | Ball & Stick           | Element symbol     | 0.3          | 1.0             |
| Single Ions               | =ion=      | Spacefill              | Element symbol     | 0.1          | 1.0             |
| Water / Solvent           | =water=    | Gaussian Surface       | Element symbol     | 2.0          | 0.3             |

* Representation Types

Each target can use any of the following representations:

| Key                  | Label               | Extra Options                         |
|----------------------+---------------------+---------------------------------------|
| =cartoon=            | Cartoon             | Tubular helices on/off                |
| =backbone=           | Backbone            | —                                     |
| =ball_and_stick=     | Ball & Stick        | Ignore hydrogens on/off               |
| =line=               | Line                | Ignore hydrogens on/off               |
| =spacefill=          | Spacefill           | Ignore hydrogens on/off               |
| =carbohydrate=       | Carbohydrate        | —                                     |
| =putty=              | Putty               | Size theme: uniform / uncertainty     |
| =molecular_surface=  | Molecular Surface   | Ignore hydrogens on/off               |
| =gaussian_surface=   | Gaussian Surface    | Ignore hydrogens on/off               |
| =off=                | Hide / Off          | —                                     |

* Color

Each target has a color configuration with two modes:

- *Theme*: Uses a Mol* built-in coloring scheme:
  - =chain-id=: Color by chain
  - =element-symbol=: Color by atom type
  - =secondary-structure=: Color by secondary structure (proteins only)

- *Solid*: Applies a single hex color (e.g., =#ff0000= for red) to all atoms in the target.

* Size & Opacity

- *Size*: Scales the representation size. The effect depends on the representation:
  - For *Ball & Stick*, *Cartoon*, etc.: Scales the radius of atoms/bonds.
  - For *Gaussian Surface*: Sets the base radius of the surface.

- *Opacity*: Sets the transparency (0.0 = fully transparent, 1.0 = fully opaque).

* Quality

Controls the rendering quality:
- =auto=: Let Mol* choose the best quality.
- =highest=, =high=, =medium=, =low=, =lowest=: Manually set the quality level.

* Sub-Parameters

Some representations expose additional options in a collapsible drawer:

| Representation       | Option              | Effect                                                |
|----------------------+---------------------+-------------------------------------------------------|
| Cartoon              | Tubular helices     | Renders helices as cylinders instead of ribbons       |
| Ball & Stick         | Ignore hydrogens    | Hides hydrogen atoms                                  |
| Line                 | Ignore hydrogens    | Hides hydrogen atoms                                  |
| Spacefill            | Ignore hydrogens    | Hides hydrogen atoms                                  |
| Molecular Surface    | Ignore hydrogens    | Excludes hydrogens from surface calculation           |
| Gaussian Surface     | Ignore hydrogens    | Excludes hydrogens from surface calculation           |

* Applying Changes

Click the *Apply* button at the bottom of the page to save your settings. Changes are stored in your browser's synced storage and take effect the next time you open a structure.
