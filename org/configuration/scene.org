The *Canvas & Camera* section in the Options page controls the global appearance of the Mol* viewer.

* Canvas Background Color

Sets the background color of the Mol* viewer. You can:
- Click the color picker to select a color visually.
- Enter a hex color code directly (e.g., =#111111= for dark mode).

The default is white (=#ffffff=). For a quick dark mode, use the built-in *Dark Mode Canvas* preset from the extension popup.

* Camera Settings

Lock the camera to a specific position and orientation when the viewer opens. This is useful for creating reproducible figures or shared views.

Enter a JSON object with the camera state. Example:

#+begin_src json
{
  "target": [0, 0, 0],
  "position": [50, 50, 50],
  "up": [0, 1, 0]
}
#+end_src

Leave this field empty to let Mol* auto-fit the camera to the structure.

**Note**: Camera settings are applied *after* the structure is loaded, so the viewer may briefly show the auto-fit view before transitioning to your saved position.

* Applying Changes

Click the *Apply* button at the bottom of the page to save your settings. Changes are stored in your browser's synced storage and take effect the next time you open a structure.

**Note**: Label customization is now configured in [[file:custom-rules/][Custom Rules]] on a per-rule basis, rather than globally.
