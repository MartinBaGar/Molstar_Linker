Custom Rules let you highlight specific chains, residues, or atoms with tailored visual styles. These rules override the global target settings for precise control over your structures.

Click *+ Add New Rule* at the bottom of the *Custom Rules* panel to create a rule.

* Rule Modes

** Simple Mode**

Designed for quick, intuitive targeting:
- *Chain*: Target a specific chain (e.g., =A=).
- *Res Range*: Target a range of residues (e.g., =5-50= or =10-20,30-40=).
- *Specific Res*: Target specific residues (e.g., =10,15,20=).
- *Atom*: Target atoms by name (e.g., =CA= for alpha carbon).
- *Element*: Target atoms by element (e.g., =FE= for iron).
- *Atom Idx*: Target atoms by index (e.g., =100,101,102=).

** Expert Mode**

For advanced users, this mode accepts raw JSON selectors for full control over the Mol* selection language. Example:

#+begin_src json
{
  "auth_asym_id": "A",
  "auth_seq_id": 50
}
#+end_src

* Representation Options

| Field      | Description                                                                                     |
|------------+-------------------------------------------------------------------------------------------------|
| Style      | Choose between:
            - *Color Highlight Only*: Overlays a color on the existing representation.
            - *Spawn: [Representation]*: Creates a new representation (e.g., Ball & Stick, Surface). |
| Color      | Theme or solid color (same as global targets).                                                  |
| Size       | Scales the representation size. Effect depends on the representation type.                     |
| Opacity    | Sets transparency (0.0 = fully transparent, 1.0 = fully opaque).                                |
| Quality    | Controls rendering quality (auto, highest, high, medium, low, lowest).                         |

* Sub-Parameters

Some representations expose additional options:

| Representation       | Option              | Effect                                                |
|----------------------+---------------------+-------------------------------------------------------|
| Cartoon              | Tubular helices     | Renders helices as cylinders instead of ribbons       |
| Ball & Stick         | Ignore hydrogens    | Hides hydrogen atoms                                  |
| Line                 | Ignore hydrogens    | Hides hydrogen atoms                                  |
| Spacefill            | Ignore hydrogens    | Hides hydrogen atoms                                  |
| Molecular Surface    | Ignore hydrogens    | Excludes hydrogens from surface calculation           |
| Gaussian Surface     | Ignore hydrogens    | Excludes hydrogens from surface calculation           |

* Storytelling Options

These options add annotations to enhance presentations or publications:

| Field               | Effect in Mol*                                                                   |
|--------------------+----------------------------------------------------------------------------------|
| Floating 3D Label  | Renders a text label anchored to the selection.                                  |
| Text Size           | Sets the size of the label text.                                                 |
| Text Color          | Sets the color of the label text.                                                |
| Border Size         | Sets the width of the label border.                                             |
| Border Color        | Sets the color of the label border.                                             |
| Hover Tooltip       | Shows a text badge when the user hovers over the selection.                     |
| Focus Camera        | Auto-zooms the view to the selection when the viewer opens.                     |

* Rule Order

Rules are applied in the order they appear in the list. You can drag and drop rules to reorder them.

* Performance & Security Limits

To ensure smooth performance and security:

- *Rule Cap*: You can define up to 50 custom rules.
- *Text Limits*: Labels are truncated at 256 characters, and tooltips at 512 characters.
- *Sanitization*: The extension validates selectors to ensure they use valid syntax.

* Example: Highlight the Active Site

To highlight residues 100–110 in chain A with an orange color and a label:

1. *Rule Name*: Active Site
2. *Mode*: Simple
3. *Numbering*: auth (or label, depending on your structure)
4. *Chain*: A
5. *Res Range*: 100-110
6. *Style*: Color Highlight Only
7. *Color*: Solid → =#ff6600=
8. *Floating 3D Label*: Active site
9. *Focus Camera*: On
