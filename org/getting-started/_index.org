Welcome to the official documentation for *Mol* Linker*!

* Overview
Mol* Linker is a browser extension that seamlessly integrates with GitHub, GitLab, RCSB, and other structural biology repositories. It automatically detects molecular structure files (e.g., =.pdb=, =.cif=, =.gro=) and injects a one-click badge to open them in the [[https://molstar.org/viewer/][Mol* web viewer]] — rendered natively with your preferred visual styles.

* Key Features
- *Automatic Detection:* Scans web pages for structure files and injects interactive badges.
- *Native Mol* Rendering:* Uses Mol*'s high-performance engine for smooth, detailed visualization.
- *Customizable Scenes:* Configure representations, colors, and transparency for each molecular component.
- *Custom Rules:* Highlight specific chains, residues, or atoms with tailored visual styles.
- *Presets:* Save and apply your favorite configurations across all structures.
- *Security & Privacy:* Files are fetched securely and rendered locally in a sandboxed environment.
- *Custom Domains:* Authorize private or self-hosted GitLab instances for seamless integration.
